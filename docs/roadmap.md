# KinderHub Roadmap

## Current Phase: MVP Development

### Q3 2025 — Foundation

**Focus:** Core platform functionality

- [x] Project setup (Kotlin Multiplatform)
- [x] Authentication system (Auth0)
- [x] Basic UI components and theming
- [x] Activity discovery screens
- [x] Activity detail view
- [x] Booking flow
- [x] User account management
- [x] Internationalization (EN, FR, GA)
- [ ] Backend API integration
- [ ] Payment processing
- [ ] Push notifications

### Q4 2025 — Provider Tools

**Focus:** Enable providers to list and manage activities

- [ ] Provider registration and onboarding
- [ ] Activity creation and management
- [ ] Availability calendar
- [ ] Booking management dashboard
- [ ] Provider messaging
- [ ] Payout system

### Q1 2026 — Growth Features

**Focus:** Enhance user experience and retention

- [ ] AI-powered activity recommendations
- [ ] Advanced search and filters
- [ ] Reviews and ratings
- [ ] Favorites and wishlists
- [ ] Calendar sync (Google, Apple)
- [ ] Family/child profiles

### Q2 2026 — Scale

**Focus:** Platform growth and optimization

- [ ] Provider analytics dashboard
- [ ] Marketing tools for providers
- [ ] Referral program
- [ ] Subscription/membership options
- [ ] Performance optimization
- [ ] Additional language support
